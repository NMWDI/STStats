Plot locations directly in QGIS

  1. Add a Vector layer
  2. Check Protocol to HTTP(s),cloud,etc
  3. Set Type to "GeoJSON"
  4. Set URI to `https://st2.newmexicowaterdata.org/FROST-Server/v1.1/Locations?$filter=properties/agency eq 'NMBGMR'&$resultFormat=geojson`
